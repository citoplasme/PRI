Abstract

Sendo este o terceiro trabalho semanal da UC de PRI e, tendo já sido lecionadas técnicas para associar stylesheets a ficheiros XML, este tema é o núcleo deste projeto.

Sendo assim, foi proposta, numa primeira fase, a realização de um project record, bem como o seu XSL (eXtensible Stylesheet Language for Transformation). Através deste stylesheet tornou-se possível organizar o visual do project record, permitindo uma simples leitura do mesmo.

Além disso, este projeto requere a realização de uma stylesheet para um dataset sobre arqueossítios. Note-se que a principal diferença desta tarefa comparativamente com a referente ao project record é que esta implica a geração de um índice e, consequentemente, uma página web por entrada no ficheiro. Quer isto dizer que é necessário percorrer a árvore de parser uma primeira vez para gerar o índice e outra para as páginas individuais.
