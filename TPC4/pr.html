Abstract

Este quarto trabalho semanal baseia-se na criação de um servidor em nodeJS referente à gestão de um arquivo sonoro, sendo que este deve responder a diferentes pedidos de um cliente.

O primeiro passo foi dividir o dataset que continha todos os documentos sonoros em vários ficheiros, cada um contendo um e um só documento sonoro. Além disso, cada um destes novos ficheiros teve associado ao seu nome um identificador, utilizado para interrogar o servidor, mais tarde.

Em seguida, foi definida uma stylesheet que converte um documento sonoro XML numa página HTML, dispondo todo o seu conteúdo de forma percetível e simples.

No que toca ao servidor, este estaria a funcionar, recebendo pedidos na forma /musica/NUM, sendo que procura o valor presente na variável NUM, retornando a informação presente no documento identificado pelo mesmo. Note-se que o documento XML é obtido primeiramente, sendo seguida a obtenção da stylesheet de conversão para HTML e, por fim, do ficheiro de CSS associado ao styling do documento web. É, ainda, de destacar que o servidor apenas responde a pedidos de GET, sendo que retorna erro quando o método ou o pedido não são suportados.
