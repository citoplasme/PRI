Abstract

Este último trabalho semanal baseia-se na criação de um servidor em NodeJS com auxílio de ExpressJS referente à gestão de uma base de dados em MongoDB. Assim, com auxílio de roteadores e uma API de dados, sejam efetuadas operações de listagem sobre a BD, permitindo um aprofundamento na matéria em questão.

O servidor apenas poderia receber pedidos de GET para /obras, listando todas as obras na BD; opcionalmente, este pedido poderia receber uma filtragem por período a partir de /obras?periodo=X, devolvendo apenas as obras do período pretendido; também poderia ser /obras?anoCriacao=Y, devolvendo todas as obras criada no ano Y; e, para o caso de ser pretendido efetuar uma listagem de todas as obras de um compositor, o pedido deveria ser /obras?compositor=A.

Além disso, o servidor poderia receber mais dois pedidos: /compositores, devolvendo todos os compositores (valores distintos) existentes na BD; e /compositores/:nome, retornando os mesmos que valores que no caso do pedido /obras?Compositor=A, sendo que A e nome deveriam ter o mesmo valor.

Por fim, note-se que não foram utilizadas vistas, já que o objetivo do trabalho era aprofundar a definição de APIs de dados.
