Abstract

Este trabalho semanal baseia-se na criação de um servidor em NodeJS com auxílio de ExpressJS referente à gestão de uma base de dados em MongoDB, com interação com o File System. Assim, com auxílio de roteadores e uma API de dados, sejam efetuadas operações de listagem sobre a BD, permitindo um aprofundamento na matéria em questão.

O servidor deveria permitir a adição de multiplos ficheiros ou apenas um ao sistema, bem como listar todos os que se encontram na BD. Além disso, ao efetuar um click sobre um registo, devem ser apresentadas as informações do mesmo, dentro de um modal. No caso desse registos ser referente a uma imagem, esta deve ser apresentada.

Note-se que as pastas referentes a conteúdo foram colocadas vazias no repositório devido ao peso a elas associado. Além disso, tendo em conta a necessidade de representação das imagens, estas devem ser guardadas na diretoria data e em public/images.
